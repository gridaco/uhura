# Surfaces

- **Status:** Stable router and working taxonomy
- **Lifetime:** Stable navigation; classification is revisable
- **Authority:** Working classification only

This directory classifies language-managed presentation layers and
orchestration primitives that sit outside the ordinary element tree.
Version-scoped surface entries must separate semantic lifecycle from
renderer-owned presentation, focus mechanics, and motion.

Version-scoped surface pages live with the current draft:

- [v0 `sheet`](../drafts/v0/surfaces/sheet.md) — the implemented default
  modality and its structural dismiss lifecycle.
- [v0 `dialog`](../drafts/v0/surfaces/dialog.md) — a named modality with no
  accepted design or realization yet.
