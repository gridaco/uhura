# Integrations

Integration is a cross-cutting facet for capabilities backed by a renderer,
host driver, external platform, or service. A concrete capability remains filed
under its primary form and links to any shared integration contract documented
here.

Integration documents must make availability, capabilities, effects, privacy
and security boundaries, fallback behavior, and offline or deterministic Editor
representation explicit.
