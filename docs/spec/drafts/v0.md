# Uhura v0 historical pointer

- **Status:** Retired UI-first incubation experiment
- **Authority:** Historical provenance only
- **Successor baseline:** Uhura 0.3
- **Active successor design:** [Uhura 0.4 incubation candidate](0.4/)

The former v0 draft described the UI-first store, view, renderer, and project
topology used by the first Instagram implementation spike. That topology was
superseded by the machine-first Uhura 0.3 engine and is not a base
specification for later language generations.

Git history retains the exact former draft. Current source-comment,
declaration-documentation, and markup-annotation decisions belong to
[RFC 0003](../../rfcs/0003-source-comments-docs-and-annotations.md), not to the
retired v0 syntax.

No later version inherits v0's grammar, runtime tuple, file kinds, widget
names, or implementation layout unless it restates them explicitly.
